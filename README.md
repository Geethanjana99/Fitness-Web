# Fitness Sports Center

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Setup Instructions

1. Install the necessary dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
